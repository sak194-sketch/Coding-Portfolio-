# Coding-Portfolio

A collection of six independent coding projects from various research and academic work, 
written in Stata. Each `.do` file represents a self-contained piece of analysis,
covering data cleaning, visualization, descriptive statistics, simulation, and causal 
inference methods (instrumental variables, heterogeneity testing, LASSO).

## Projects

- **BGD data cleaning and time series graph.do** — Cleans and visualizes minimum wage data across ~50 
  Industries in Bangladesh are used in the World Bank development research group analysis.
- **Baseline merging and analysis.do** — Survey data analysis on political participation, 
  education attitudes, and civic engagement in Punjab, Pakistan (CERP research).
- **Simulations.do** — Monte Carlo simulations exploring statistical power, 
  Type I/II error, and confidence interval construction.
- **Reweighed.do** — Constructs inverse probability weights to correct 
  for survey attrition in a longitudinal study on corporal punishment outcomes. This is part of the CGD research. 
- **pe het test final.do** — Instrumental variable, heterogeneity, and 
  LASSO control selection analysis from the same longitudinal study.This is part of the CGD research. 
- **pe iv v3.do** — This is part of the CGD research and the do file includes Instrumental Variable Analysis.
- **pe OLS FINAL.DO** - This is part of the CGD research. It looks at the OLS analysis 

Data files are not included to protect respondent privacy.
